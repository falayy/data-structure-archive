## ðŸŽ¯ Common Patterns

### Two Pointers
- Used when array is sorted or can be sorted
- Two indices moving toward/away from each other
- Examples: Two Sum II, Container With Most Water

### Sliding Window
- Subarray/substring problems
- Window expands and contracts
- Examples: Maximum Subarray, Longest Substring

### Hash Map
- O(1) lookup for seen elements
- Trade space for time
- Examples: Two Sum, Contains Duplicate

### Kadane's Algorithm
- Maximum subarray problems
- Dynamic programming approach
- Track current and global maximum
